# Juliet_Test_Suite

A collection of test cases in the Java language. It contains examples for 112 different CWEs. 
The test suite is taken from [NIST website](https://samate.nist.gov/SRD/testsuite.php)


## Plugin 

```
SonarLint
```

## IDE

```
IntelliJ
```
